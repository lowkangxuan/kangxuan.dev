## Background
Back in 2022, I built my first ever [portfolio website](https://lowkangxuan.github.io/) using Eleventy (11ty) and Bootstrap. It was a testament to my web development skills at the time,
and honestly, I was proud of how it turned out.

Fast-forward to today, I challenged myself to rebuild my portfolio using modern frameworks like React and Tailwind CSS.
My goal was to maintain a minimalist design, similar to my previous portfolio website, and
sections to showcase my skills, experience, and projects while still having the flexibility to write blog posts like this one.

Here's a small breakdown on the technology behind the site and how much it costs me to build it.

## Tech Stack
- Tanstack Start
- Tailwind CSS v4
- Shadcn/ui

## Features
- Minimalist design
- Blog updates
- Light & Dark mode
- Search Bar for quick-search purposes
- Live clock of my timezone (GMT +8)
